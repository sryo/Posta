# posta.

Email, sorted.

A card-based Gmail and Calendar client. Organize your inbox into columns using search queries.

---

**Calendar cards** — Add cards with `calendar:today`, `calendar:week`. Filter by organizer, location, or attendees.

**Batch reply** — Select multiple threads, reply to all at once.

**Quick actions** — Hover to archive, star, reply, or delete.

**Multiple accounts** — Connect all your Gmail accounts. Switch instantly.

---

## Getting Started

```bash
npm install
npm run tauri dev
```

## Tech

SolidJS · Rust · Tauri · SQLite · Gmail API · Google Calendar API
